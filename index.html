This is a branch of the existing randomizer that implements a lot of new features including:

  • Worlds, spells, trinities, summons, and stat ups are all items and thus can be placed anywhere
  • Many checks that were previously un-randomizable or were only shuffled are now fully randomized, including:
    • Atlantica clams
    • Static item events (Crystal Trident, Forget-Me-Not, Postcards, etc)
    • All dalmatian rewards
    • All Geppetto and Pinocchio rewards
    • All synthesis recipes
    • Map prizes (Trinities that only drop item bubbles, Wonderland chairs and flowers, etc)
  • Fully GUI based settings/seed/mod zip generation with full OpenKH integration.
  • Recent additions of original randomizer features like Destiny Islands finish are now here!
  • Options for keyblade stat randomization or full shuffle, including hidden stats like crit rate, crit str bonus, and recoil, fully displayed in the weapon description.
  • Randomized cups with options to exclude no cups, all cups, or only Hades cup
  • Addition of Final Rest Door Key, an item which makes the door in Final Rest appear
  • Addition of Lucky Emblems, with settings to decide how many unlock End of the World and the Final Rest Door Key
  • Tons of works stabilizing various crashes, such as revisiting incomplete worlds after Riku Ansem, warping out of worlds at times that would normally cause issues, sequence break soft locks, etc
  • Option to shorten the final fights - teleporting Sora to the final cutscene after defeating Ansem 1 > Darkside > Ansem 2
  • EXP multiplier options which change how much exp is needed to level instead of how many is acquired (making tech point farming a bit more viable)
  • Full Archipelago support for multiworld randomizers
