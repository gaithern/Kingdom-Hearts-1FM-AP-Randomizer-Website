  1. Install gaithern/KH1-LUA-LIBRARY in OpenKH.
  2. Install gaithern/KH1-GEOGUESSER in OpenKH.
  3. In game, to find coordinates press L2 + L1 + R2 + R1 + Triangle to show your current Room + World + X Y Z.
  4. Enter the coordinates for the locations you want players to find in the form below.
  5. Click "Generate & Download Find Script" and share the file with your friends.

HOW TO FIND COORDINATES:

  1. Install gaithern/KH1-LUA-LIBRARY in OpenKH.
  2. Install gaithern/KH1-GEOGUESSER in OpenKH.
  3. Install the script provided by the person who set the coordinates.
  4. A text box will pop up and a sound will play when they are within range of your coordinates.
